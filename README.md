## eslint

eslint: https://cn.eslint.org/

安装： 
npm i -D eslint

eslint init 进行配置eslintrc.js

忽略单行检查
// eslint-disable-line

忽略多行检查
/* eslint-disable */ 和 /* eslint-enable */ 组合使用

忽略整个文件或者以下代码全部忽略
/* eslint-disable */ 写在文件头部或写在需要忽略的代码上方
